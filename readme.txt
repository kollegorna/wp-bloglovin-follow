== Description ==

This plugin allows the user to display their Bloglovin Follow button in different contexts and/or in a widget.

The Bloglovin Follow code can be [obtained here](http://www.bloglovin.com/widgets/).

[Source code and development on GitHub](http://github.com/kollegorna/wp-bloglovin-follow)

This plugin is sponsored by [Rodeo Magazine](http://rodeo.net) and [Kollegorna](http://www.kollegorna.se).
Much of the code and inspiration came from the [Facebook Likes You!](http://wordpress.org/extend/plugins/facebook-likes-you/) plugin by Piotr Sochalewski.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Configure its settings through 'Settings' > 'Bloglovin Follow' in WordPress admin.
